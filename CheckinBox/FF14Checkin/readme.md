### 必要的环境变量

- fflogin_name # 登录号
- fflogin_password # 登录密码
- area_name # 大区(陆行鸟、猫小胖、莫古力)
- server_name # 游戏服务器(神意之地、萌芽池、拉诺西亚……)
- role_name # 游戏名(必须完全一致)
